# Metodologia

## Objetivo

Investigar situações em que pedidos são marcados como prontos para retirada antes da conclusão do processo operacional, gerando tempo de espera para entregadores.

## Etapas do Projeto

1. Observação do problema em campo.
2. Validação da dor por meio de formulário com entregadores.
3. Construção de base de dados sintética.
4. Análise exploratória utilizando SQL.
5. Construção de dashboard no Amazon QuickSight.
6. Proposta de solução utilizando IA Generativa.

## Fontes de Dados

- Respostas coletadas via Google Forms.
- Dados sintéticos gerados para simulação do cenário.
