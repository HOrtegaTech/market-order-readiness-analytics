# Metodologia da Base Sintética

A base de dados utilizada neste projeto foi gerada de forma sintética para simular cenários observados na operação de entregas de mercado.

Foram criados 500 registros contendo informações relacionadas às hipóteses definidas no projeto.

A distribuição dos dados foi planejada para representar situações como:

- Horários de pico.
- Dias de chuva.
- Datas comemorativas.
- Pedidos com diferentes quantidades de itens.
- Ocorrências de pedidos marcados como prontos antes da finalização.

O objetivo da base é permitir análises exploratórias utilizando SQL e visualizações em dashboard.
